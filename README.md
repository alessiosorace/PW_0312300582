# Sviluppo di una pagina web per il download dei report di sostenibilità di un’impresa del settore primario

Project Work per il Corso di Laurea L-31 in Informatica per le Aziende Digitali presso l'Università Telematica Pegaso. Il progetto consiste nella progettazione e sviluppo, tramite i linguaggi HTML e CSS, di una pagina web dedicata al download del report di sostenibilità di un'impresa operante nel settore primario.
